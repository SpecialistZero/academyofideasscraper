This is a simple program that allows you to scrap articles from “academyofideas.com”

In order to scrape a single article, 
1. Copy article link from website.
Note: It should look something like this “https://academyofideas.com/2021/04/guide-to-self-actualization-abraham-maslow-healthy-individual/”. 
2. Then paste it into console when prompted.

In order to scrape a category of articles:
1. click on one of the categories at the top of the page. 
2. Then copy the link from the browser. 
3. Paste link into python when prompted. 
Note: The link should look something like this “https://academyofideas.com/category/psychology/”
