This is a simple program that allows you to scrap articles from “academyofideas.com”

In order to scrape a single article, copy article link from website. It should look something like this “https://academyofideas.com/2021/04/guide-to-self-actualization-abraham-maslow-healthy-individual/” then paste it when asked.

In order to scrape a category of articles click on one of the categories at the top of the page. Then copy the link from the browser, then past it into python when prompted. The link should look something like this “https://academyofideas.com/category/psychology/”
